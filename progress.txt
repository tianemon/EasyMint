## [2026-05-28] - 任务: JSONL 流式解析器

### 完成内容:
- 创建 app/main/utils/jsonl-parser.ts：JsonlParser 类继承 EventEmitter
- start() 接受 Readable stream，监听 data/end 事件
- feed() 按行分割、JSON.parse、容错不完整行跨 chunk 拼接
- classify() 按 type 分类：system(init/status) → 系统事件、stream_event → text_delta/tool_use 增量、assistant → 文本合并 + tool_use、user → tool_result、result → usage/cost
- flush() 子进程退出时清空缓冲区
- stop() 移除 stream 监听器并清空缓冲区
- 支持 Buffer 和 string 输入
- emitRaw 选项控制是否发出未识别行
- 导出类型守卫：isAssistantEvent/isToolUseEvent/isToolResultEvent/isSystemEvent/isResultEvent
- 参考 open-design/apps/daemon/src/claude-stream.ts 的事件分类逻辑
